# airay-skills

Claude Code skills by airay for daily productivity.

## Installation

```bash
/plugin marketplace add akira82-ai/airay-skills
```

## Skills

| Skill | Description |
|-------|-------------|
| airay-agent-review | 每日工作复盘报告生成器 |
| airay-file-organizer | AI 语义分析智能文件分类工具，Johnny Decimal 编号系统 |
| airay-github-smart-commit | 智能 GitHub 提交助手，自动分析变更生成 Conventional Commits |
| airay-idea-to-post | 通过多轮对话将灵感转化为高质量文章 |
| airay-insights-zh | Claude Code insights 汉化版 |
| airay-lark-wiki-agent | 飞书知识库 CRUD 操作工具 |
| airay-skill-backup | 备份和恢复已安装的 Claude Code 全局技能 |
| airay-skill-usage | 统计分析 skill 使用情况 |
